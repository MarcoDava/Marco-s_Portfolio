ScriptSense

Timeline : December 2024 - January 2025
In this project I created a Tensorflow Handwriting Recognition Web Application. This application was made to solve documents that do not support cursor selection over text. The application is also able to accurately distinguish handwritten words. The interface is designed to be user-friendly with easy importing of pictures.



How I did it:

This project is heavily inspired by the various resources online. The model takes in the colored and white pixels, compressing the image or drawn letter into a lower resolution matrix of 1's and 0's. From this compression, the time taken to distinguish the letters becomes faster. The AI then returns what the letter best resembles. To achieve this model, I used python and tensorflow. Tensorflow provided the necessary learning models for the AI to recognize the different characters.
